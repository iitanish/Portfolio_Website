Projects

RSVT-Radio Frequency Vulnerability Tester

This project involves the development of a hardware device capable of capturing and replaying car key signals to interact with the corresponding vehicle. The system is designed to demonstrate the concept of signal replay attacks, which exploit vulnerabilities in older car keyless entry systems that do not utilize rolling code mechanisms.

Wireless Payload Execution Tool

This project involves the development of an innovative IoT device that functions as a remote keyboard. The device allows users to control a target computer by connecting to its built-in Wi-Fi module. Additionally, it supports pre-programmed scripts stored in the hardware, which are automatically executed when the device connects to the target system. The project demonstrates advanced capabilities in IoT, automation, and hardware-software integration, showcasing how IoT devices can interact with computers for automation and remote control purposes.
